
Keywords: Tobii, EyeX, Calibration, cursor control, Visual Studio 2013, .NET, C#

- It allows:
- to calibrate a Tobii eye-tracker (compatible with the library EyeX) by animated figures.
- to see the last calibration done. 
- to place an user correctly in the tracking box range of the eye-tracker
- to control the cursor by pressing the letter 'Q' or 'q'.
